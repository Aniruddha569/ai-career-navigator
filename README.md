# AI Career Navigator

An AI-powered career guidance platform that helps users analyze resumes, identify skill gaps, and prepare for interviews using Google Gemini AI.

## Features

- **Resume Analysis** — Upload your resume and get instant AI feedback
- **Skill Gap Analyzer** — Compare your skills against target job requirements
- **Interview Prep** — Practice with AI-generated interview questions
- **Dashboard** — Track your progress and recent activity

## Tech Stack

**Frontend**
- React + Vite
- CSS (custom styling)

**Backend**
- Node.js + Express
- PostgreSQL
- JWT Authentication
- Google Gemini AI API
